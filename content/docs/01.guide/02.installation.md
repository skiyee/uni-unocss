## 环境要求

在开始之前，请确保你的开发环境满足以下要求：

| 依赖              | 最低版本 | 推荐版本 |
|-------------------|----------|----------|
| Node.js           | 18.0.0   | 20.18.0  |
| pnpm / npm / yarn | -        | 最新版   |
| Vite              | 5.0.0    | 5.2+     |
| Vue               | 3.0.0    | 3.4+     |

::note
uni-unocss 专为现代 UniApp + Vite 项目设计，支持 HBuilderX 内置编译器或 CLI 项目。
::

## 安装

使用你喜欢的包管理器安装 `uni-unocss`：

::code-group
```bash [pnpm]
pnpm add uni-unocss
```

```bash [npm]
npm install uni-unocss
```

```bash [yarn]
yarn add uni-unocss
```
::

::warning
**重要**：无需安装unocss，系统将替你选择最合适UniApp项目的版本
::

## 配置 Vite

在你的 `vite.config.ts` 中添加 UnoCSS 插件：

```ts [vite.config.ts]
import Uni from '@dcloudio/vite-plugin-uni'
import UnoCSS from 'uni-unocss'
import { defineConfig } from 'vite'

export default defineConfig({
  plugins: [
    // UnoCSS 需要在 Uni() 插件之前
    UnoCSS(),
    Uni(),
  ],
})
```

## 创建 UnoCSS 配置文件

在项目根目录创建 `uno.config.ts` 文件：

```ts [uno.config.ts]
import {
  defineConfig,
  presetUni,
  transformerDirectives,
  transformerVariantGroup,
} from 'uni-unocss'

export default defineConfig({
  presets: [
    presetUni(),
  ],
  transformers: [
    transformerDirectives(),
    transformerVariantGroup(),
  ],
})
```

## 引入样式

在 `main.ts` 中引入 UnoCSS 虚拟模块：

```ts [src/main.ts]
import { createSSRApp } from 'vue'
import App from './App.vue'

// 引入 UnoCSS 样式
import 'uno.css'

export function createApp() {
  const app = createSSRApp(App)
  return {
    app,
  }
}
```

::tip
你可以使用 `uno.css` 或 `virtual:uno.css`，两者效果相同。
::

## 验证安装

安装完成后，启动开发服务器验证配置：

```bash
pnpm dev:h5
```

然后在任意页面使用原子类：

```vue
<template>
  <view class="text-red-500 text-2xl font-bold">
    Hello uni-unocss!
  </view>
</template>
```

如果文字显示为红色并加粗，说明 uni-unocss 已成功配置。

## 目录结构

一个典型的 uni-unocss 项目结构如下：

```
your-project/
├── src/
│   ├── pages/
│   │   └── index.vue
│   ├── static/
│   ├── App.vue
│   ├── main.ts
│   ├── manifest.json
│   └── pages.json
├── uno.config.ts          # UnoCSS 配置文件
├── vite.config.ts
└── package.json
```
