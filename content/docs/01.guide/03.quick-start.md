## 基础用法

安装配置完成后，你可以在任意 Vue 组件中直接使用 UnoCSS 的原子类：

```vue [pages/index.vue]
<template>
  <view class="h-screen flex items-center justify-center">
    <view class="text-center">
      <text class="text-3xl font-bold text-blue-500">
        Hello uni-unocss!
      </text>
      <text class="mt-4 text-gray-500">
        采用更激进的语法在 UniApp 中使用 UnoCSS
      </text>
    </view>
  </view>
</template>
```

## 使用图标

配合 `presetIcons` 可以使用纯 CSS 图标：

```ts [uno.config.ts]
import {
  defineConfig,
  presetIcons,
  presetUni,
} from 'uni-unocss'

export default defineConfig({
  presets: [
    presetUni(),
    presetIcons({
      scale: 1.2,
      cdn: 'https://esm.sh/',
    }),
  ],
})
```

然后在组件中使用：
```vue
<template>
  <view class="flex items-center gap-2">
    <view class="i-carbon-logo-github text-2xl" />
    <text>GitHub</text>
  </view>
</template>
```

::note
使用图标预设时，请确保安装了对应的图标集，如 `@iconify-json/carbon`。
::
